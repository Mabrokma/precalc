Exponential functions

Exponential functions have the form `f(x) = a b^x` where both `a` and `b>0` are real numbers. We call `b` the base.

Concave up

A function `f` is concave up on an interval if one of the following statements is true:
  • `f` increases at an increasing rate
  • `f` decreases at an increasing rate

    • Concave down

    A function `f` is concave down on an interval if one of the following statements is true:
  • `f` decreases at an decreasing rate
  • `f` increases at an decreasing rate
